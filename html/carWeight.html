Edit Car Weight

Requirements

  • HxD Hex Editor
  • The target cdfbin

    Where?

    yourMod/Vehicles/Physics/Chassis/yourCar.cdfbin
     file from AMS2

Step-by-step Instructions

1. Open the cdfbin

Where?

yourMod/Vehicles/Physics/Chassis/yourCar.cdfbin
 file with HxD

Launch HxD and open the cdfbin

Where?

yourMod/Vehicles/Physics/Chassis/yourCar.cdfbin
 file related to the car you want to edit.

2. Search for the specific hex pattern

Use Ctrl + F and set the search type to Hex-values and set the Search Direction to All. Then search for one of the following patterns:

22 67 0B 57 AB Float
or
21 67 0B 57 AB Integer

These patterns may vary depending on the version of the file or vehicle.

3. Modify the weight (KG) value

After finding the correct pattern, locate the float value that represents the weight, which follows the pattern. Modify it accordingly.

The weight is stored in little-endian float format. For example:

00 00 48 42 = 50.0 KG
00 00 96 43 = 300.0 KG

4. Save the changes

After editing the value, go to File > Save or press Ctrl + S to save your modifications.
